# 🎬 Movie Ticket Booking System

A simple desktop application developed using Python, Tkinter, and SQLite for booking movie tickets.

## Features

- User-friendly GUI
- Book movie tickets
- Store booking details in SQLite database
- View booking information
- Simple and easy-to-use interface

## Technologies Used

- Python
- Tkinter
- SQLite

## How to Run

1. Install Python.
2. Download or clone this repository.
3. Open the project in VS Code.
4. Run:

```bash
python movie_ticket_booking.py
```
